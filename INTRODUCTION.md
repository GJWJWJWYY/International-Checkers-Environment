## 游戏简介

### 什么是国际跳棋？

国际跳棋（International Draughts/Checkers）是一种双人对弈的策略棋类游戏，起源于欧洲，特别流行于法国、荷兰等国家。与英美跳棋（8×8棋盘）不同，国际跳棋使用**10×10棋盘**，规则更加复杂，战术更加丰富。

### 棋盘与棋子

- **棋盘**：10行×10列，共100格
- **可用格子**：只使用深色格子（50格）
- **棋子数量**：每方20个棋子
- **棋子颜色**：白色（底部）和黑色（顶部）

## 基本规则

### 1. 回合制

- **白方先手**：白色棋子（底部）先行
- **交替行棋**：双方轮流移动一个棋子
- **每次一步**：每个回合只能移动一个棋子（除非连续吃子）

### 2. 移动方向

#### 普通棋子
- **只能向前移动**：白方向上（第0行方向），黑方向下（第9行方向）
- **斜向移动**：只能斜向移动到相邻的深色格子
- **每次一格**：普通移动时每次只能移动一个对角格

#### 王棋（升级后）
- **任意方向**：可以向前、向后、向左斜、向右斜四个对角线方向移动
- **飞行距离**：可以移动任意格数（直到遇到棋子或边界）

### 3. 成王

当棋子到达对方底线时，自动升级为**王棋**（King）：

- **白方成王**：到达第0行（黑方底线）
- **黑方成王**：到达第9行（白方底线）
- **标记**：王棋在界面上会显示皇冠图标

## 移动规则

### 普通移动

当**没有吃子机会**时，可以进行普通移动：

1. **方向限制**：普通棋子只能向前斜向移动
2. **距离限制**：每次移动一个对角格
3. **目标必须为空**：只能移动到空的深色格子

**示例**：
```
移动前：          移动后：
  ○                 
    ·         →       ○
```

### 王棋移动

王棋可以**飞行**：

1. **四个方向**：前后左右四个对角线方向
2. **任意距离**：可以移动1格、2格、3格...直到棋盘边界
3. **遇到棋子停止**：不能穿过其他棋子（吃子时除外）

**示例**：
```
  ♔           可以移动到任何标记为 ✓ 的位置
    ✓
      ✓
        ✓
```

## 吃子规则

### 核心原则

国际跳棋的吃子规则是游戏的核心，有**三大强制规则**：

#### 1️⃣ **强制吃子**
如果场上有吃子机会，**必须吃子**，不能选择普通移动。

#### 2️⃣ **强制吃最多**
如果有多个吃子选择，**必须选择能吃最多棋子的路径**。

#### 3️⃣ **连续吃子**
吃掉一个棋子后，如果还能继续吃，**必须继续吃**，直到无法再吃为止。

### 普通棋子吃子

#### 基本吃子

- **跳过敌子**：跳过相邻的敌方棋子，落在后面的空格
- **移除敌子**：被跳过的敌子从棋盘上移除
- **可以后向吃**：普通棋子可以向后跳吃（但不能后退普通移动）

**示例**：
```
吃子前：              吃子后：
  ○                     
    ●         →           ○  （●被吃掉）
      ·
```

#### 连续吃子

一次移动中可以连续吃多个棋子：

**示例**：
```
  ○                           ○
    ●                     
      ·       →             
        ●                   
          ·           

连续吃掉两个棋子：○ → 中间位置 → 最终位置
```

### 王棋吃子

王棋的吃子更加灵活和强大：

#### 1. **飞行吃子**
- 可以从任意距离飞越敌子
- 可以落在敌子后面的任意空格（不限距离）

**示例**：
```
  ♔                       
    
      ●
        
          ✓
            ✓
              ✓    可以落在 ● 后面的任意空格
```

#### 2. **连续吃子 + 换方向**
- 可以连续吃多个棋子
- 每次吃子后可以改变方向
- 形成"之字形"吃子路径

**示例**：
```
  ♔                   第一次吃子
    ●                     ↓
      ·                   ♔
        ·       →           ●
          ●                   ·  第二次吃子（换方向）
```

#### 3. **不能重复吃**
- 同一个棋子在一次移动中只能被吃一次
- 但可以多次经过同一个空格

## 王棋规则

### 成王时机

棋子到达对方底线时**立即**成王，即使是在连续吃子过程中。

### 成王后的能力

| 能力 | 普通棋子 | 王棋 |
|------|---------|------|
| 向前移动 | ✅ 1格 | ✅ 任意格 |
| 向后移动 | ❌ 不能 | ✅ 任意格 |
| 向前吃子 | ✅ 跳1格 | ✅ 飞行吃子 |
| 向后吃子 | ✅ 跳1格 | ✅ 飞行吃子 |
| 连续吃子 | ✅ 可以 | ✅ 可以 |
| 换方向吃 | ✅ 可以 | ✅ 可以 |
| 落点选择 | ❌ 固定 | ✅ 任意空格 |

### 王棋的战术价值

- **控制力强**：可以控制整条对角线
- **吃子灵活**：可以从远距离发动攻击
- **防守优势**：可以快速回防
- **连跳能力**：可以形成复杂的连续吃子

## 胜负判定

### 获胜条件

满足以下**任一条件**即可获胜：

1. **吃光对方所有棋子**
2. **对方无合法移动**（被困死）

### 平局条件

满足以下**任一条件**则判平局：

#### 1️⃣ **50步规则**（FMJD国际标准）

连续50步（双方各25回合）内：
- 没有吃掉任何棋子
- 没有任何棋子成王

则自动判和。

**示例**：
```
第1步：白方移动普通棋子（计数器 = 1）
第2步：黑方移动普通棋子（计数器 = 2）
...
第49步：白方移动王棋（计数器 = 49）
第50步：黑方移动王棋（计数器 = 50）→ 判和
```

**重置情况**：
- 任何一方吃子 → 计数器归0
- 任何棋子成王 → 计数器归0

#### 2️⃣ **三次重复**

相同局面（包括**轮到谁下**）出现3次，自动判和。

**注意**：必须是完全相同的局面：
- 所有棋子位置相同
- 所有棋子类型相同（普通/王）
- **轮到同一方下棋**

## 人类操作指南

### 启动游戏

#### 方法1：运行GUI版本
```bash
python checkers_game.py
```

#### 方法2：在代码中启动
```python
from checkers_game import main
main()
```

### 基本操作

#### 1. **选择棋子**

**鼠标左键点击**自己的棋子：
- 棋子会被高亮显示
- 所有可移动位置会显示绿色圆圈
- 如果有吃子机会，只显示吃子位置（强制吃子）

**提示**：
- 只能选择己方棋子（白色）
- 只能选择有合法移动的棋子

#### 2. **执行移动**

**点击目标位置**（绿色圆圈）：
- 棋子自动移动到目标位置
- 如果是吃子，被吃的棋子会被移除
- 自动切换到对方回合

#### 3. **取消选择**

**点击其他己方棋子**或**空白区域**：
- 取消当前选择
- 可以重新选择其他棋子

### 多路径选择

当一个棋子有**多条路径到达同一位置**时（都吃相同数量的棋子），需要手动选择路径：

#### 场景识别

点击目标位置后，如果：
- 顶部显示 "**Path 1/3 - Click again to cycle**"
- 用**红色圆圈**标记即将被吃的棋子
- 用**黄色圆圈**标记目标位置

说明进入了**路径选择模式**。

#### 操作方式

**方法1：空格键切换**
```
按下 [空格键] → 切换到下一条路径
红色圆圈位置会改变，显示不同路径会吃掉的棋子
```

**方法2：再次点击目标**
```
点击 [目标位置] → 确认当前显示的路径
执行移动并吃掉显示的棋子
```

**方法3：取消选择**
```
点击 [其他位置] → 退出路径选择模式
可以重新选择棋子或选择其他移动
```

#### 完整示例

```
情况：王棋从(5,5)可以到达(8,8)，有3条路径都吃2个棋子

步骤1：点击王棋(5,5)
     → 显示所有可移动位置

步骤2：点击目标(8,8)
     → 进入路径选择模式
     → 显示 "Path 1/3"
     → 红色圆圈标记路径1会吃的棋子（比如A和B）

步骤3a：按空格键
     → 显示 "Path 2/3"
     → 红色圆圈改为路径2会吃的棋子（比如C和D）
     → 再按空格切换到 "Path 3/3"

步骤3b：再次点击(8,8)
     → 确认当前路径
     → 执行移动，吃掉显示的棋子
     → 完成回合
```

### 操作技巧

#### 1. **查看可移动位置**

点击棋子后，仔细观察所有绿色圆圈：
- **大绿圈**：可以移动到这里
- **红色圆圈**：表示吃子路径（在路径选择模式下）
- **黄色圆圈**：表示目标位置（在路径选择模式下）

#### 2. **判断强制吃子**

如果选择棋子后：
- 只显示吃子位置（没有普通移动的绿圈）
- 说明有强制吃子

如果多个棋子都能吃：
- 只能选择吃得最多的那些棋子
- 吃得少的棋子无法选中

#### 3. **多路径选择策略**

当有多条路径时：
- **慢慢切换**：用空格键查看每条路径
- **观察红圈**：看清楚每条路径吃掉哪些棋子
- **考虑后果**：选择对己方最有利的路径
- **确认后点击**：想好后再点击目标确认

#### 4. **连续吃子**

如果吃掉一个棋子后还能继续吃：
- 系统会自动保持你的棋子为选中状态
- 继续显示可以继续吃的位置
- 必须继续吃，不能停止

## 进阶技巧

### 战术要点

#### 1. **控制中心**

占领棋盘中心（第4-5行）的重要性：
- 控制更多对角线
- 有更多移动选择
- 防守和进攻都灵活

#### 2. **成王时机**

何时应该努力成王：
- ✅ 己方棋子数量优势时
- ✅ 对方防线薄弱时
- ❌ 可能被吃掉时不要冒险

#### 3. **王棋运用**

王棋的正确使用方式：
- **控制长对角线**：站在对角线上控制全局
- **远程威胁**：保持距离，避免被围困
- **连跳机会**：寻找连续吃子的机会
- **回防及时**：王棋可以快速回防

#### 4. **吃子策略**

面对多个吃子选择时：
- **优先吃王**：如果能吃掉对方王棋，优先考虑（虽然代码只看数量，但战术上很重要）
- **破坏对方结构**：选择能破坏对方防线的吃子路径
- **避免反吃**：吃子后要考虑是否会被反吃
- **创造连跳**：选择能继续吃子的路径

#### 5. **防守技巧**

保护自己的棋子：
- **互相保护**：两个棋子站在一起难以被吃
- **守住底线**：不要让对方轻易成王
- **留后手**：不要把所有棋子都推上前线
- **封锁王棋**：用多个棋子限制对方王棋的活动

### 常见陷阱

#### ❌ 陷阱1：贪吃失子

```
情况：可以吃掉1个普通棋子，但之后会被吃掉2个

错误：只看眼前，吃了1个
正确：考虑后续，放弃这个吃子机会（如果有其他选择）
```

#### ❌ 陷阱2：成王陷阱

```
情况：可以成王，但成王后会立即被吃

错误：冲上去成王
正确：等待更好的时机
```

#### ❌ 陷阱3：忽视强制吃子

```
情况：己方有吃子机会，但想走别的棋

错误：试图普通移动（系统不允许）
正确：必须执行吃子
```

### 开局原则

好的开局应该：
1. **控制中心**：向中心移动棋子
2. **保持平衡**：不要过度集中一侧
3. **避免弱点**：不留单个孤立的棋子
4. **灵活应变**：根据对方走法调整

### 中局策略

中局是决定胜负的关键：
1. **寻找吃子机会**：主动创造吃子机会
2. **兑子判断**：优势时可以兑子，劣势时避免兑子
3. **成王竞赛**：争取比对方先成王
4. **空间控制**：限制对方棋子的移动空间

### 残局技巧

当棋子数量较少时：
1. **王对王**：保持距离，寻找对方失误
2. **王对棋子**：王棋通常占优，逐步围困对方
3. **50步规则**：注意计数，避免自动判和
4. **三次重复**：如果劣势，可以制造重复局面求和

## 常见问题

### Q1: 为什么我的棋子点击后没有反应？

**可能原因**：
- 不是你的回合（检查底部显示的Turn）
- 这个棋子没有合法移动
- 游戏已经结束

### Q2: 为什么只显示吃子位置，不能普通移动？

**答案**：强制吃子规则 - 有吃子机会时必须吃子。

### Q3: 为什么只有部分棋子能选中？

**答案**：强制吃最多 - 只能选择能吃最多棋子的那些棋子。

### Q4: 路径选择时如何知道选哪条？

**方法**：
1. 用空格键查看每条路径的红色圆圈
2. 比较每条路径吃掉的棋子位置
3. 选择对自己最有利的路径

### Q5: 连续吃子时能停下来吗？

**答案**：不能！必须继续吃，直到无法再吃为止。

### Q6: 王棋可以移动多远？

**答案**：任意距离，直到遇到棋子或棋盘边界。

### Q7: 50步规则如何计算？

**答案**：
- 每走一步，计数器+1
- 吃子或成王后归0
- 达到50自动判和

### Q8: 如何退出游戏？

**答案**：点击窗口的关闭按钮（X）。
